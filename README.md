# Student Registration System

A simple and interactive **Student Registration System** built using **HTML**, **CSS**, and **JavaScript**.  
This project allows users to **add**, **view**, **edit**, and **delete** student records in a clean web interface.

---

## Features

- Add new student details (Name, ID, Class, Roll No)
- Edit existing student information
- Delete unwanted student records
- Data dynamically displayed in a table
- Clean and responsive UI design

---

## Technologies Used

- **HTML5** — Structure of the application  
- **CSS3** — Styling and layout  
- **JavaScript (ES6)** — Logic for adding, editing, and deleting students  

---

## Project Structure

```
├── index.html       # Main HTML file
├── style.css        # Styles for layout and design
└── script.js        # JavaScript for functionality
```

---

## How to Run

1. Download or clone this repository:
   ```bash
   git clone https://github.com/yourusername/student-registration-system.git
   ```
2. Open the `index.html` file in any modern browser.
3. Start adding student details and manage records dynamically!
